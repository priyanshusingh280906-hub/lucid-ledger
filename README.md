# 💰 Lucid Ledger
Lucid Ledger is a sophisticated personal finance tracker designed for clarity and ease of use. It features a real-time dashboard, dynamic data visualization through Chart.js, and a mobile-responsive layout.

---

📂 Project Structure
This project follows a clean, modular structure for easy navigation and future scalability.


---

## ✨ Features

**Real-Time Financial Dashboard**

  **•  Dynamic Calculations:** Automatically updates Total Spent, Monthly Spending, and Remaining Budget as you add expenses.

  **•  Visual Analytics:** Integration with Chart.js provides a doughnut chart breakdown of spending by category (Food, Housing, Transportation, etc.).

  **•  Notification System:** Built-in UI for budget alerts and financial reminders.

**Interactive UI/UX**

  **•  Responsive Design:** Uses CSS Grid and Flexbox to ensure a seamless experience on mobile, tablet, and desktop.

  **•  Elegant Styling:** A modern "Deep Teal" and "Slate" palette with a clean glassmorphism-inspired background.

  **•  Transaction History:** A dedicated feed to track recent spending entries instantly.

---

## 🏗️ Project Phases
To ensure a robust and scalable application, Lucid Ledger is being developed in three distinct stages:

### Phase 1: Front-end & UI/UX (Current)
**• Visual Dashboard:** Instant overview of total balance, income, and expenses.

**• Transaction Tracking:** A dynamic, real-time list for logging and reviewing financial activities.

**• Responsive Design:** Optimized for a seamless experience on mobile, tablet, and desktop.

**• Modern UI:** Built with a professional yellow-and-white themed palette for high readability.

### Phase 2: Backend & Database (Upcoming)
**• Persistent Storage:** Integrating a Database (like MongoDB) to save transaction history permanently.

**• Server-Side Logic:** Developing a Node.js/Express server to manage data flow and API requests.

### Phase 3: Security & Advanced Analytics (Future)
**• User Accounts:** Implementing secure Authentication for personalized user ledgers.

**• Data Insights:** Adding visual charts and monthly spending reports.
